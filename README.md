# 🌍 geo-quiz

世界の白地図で、国の場所と名前を当てるシンプルなウェブクイズです。
ビルド不要・依存なしの静的サイトで、GitHub Pages 上で動きます。

## 🔗 アクセス

**👉 https://airpocket-soundman.github.io/geo-quiz/**

スマホ・タブレット・PC どのブラウザからでもプレイできます。

---

## 🎮 遊び方

3つのモードから選んで開始します。

### 国名 → 場所
画面上部に表示された国名を、地図でタップして答えます。
タップした国は青色でハイライトされます。納得したら **「確定」** を押して判定。

### 場所 → 国名
地図上で赤色に塗られた国を、下に並ぶ4つの選択肢から選びます。
選択肢を選ぶと青色で示されます。**「確定」** を押すまでは選び直し可能。

### 自由探索
出題なし。地図上で国をタップ／ホバーすると国名が表示されます。

## 📜 ルール

| 設定項目 | 選択肢 |
|---|---|
| 地域 | 全世界 / アジア / ヨーロッパ / アフリカ / 北米 / 南米 / オセアニア |
| 問題数 | 5問 / 10問 / 20問 |
| 採点 | 正解 +1 ／ 不正解 0（減点なし） |

- 同地域から優先的に紛らわしい誤答候補を出題します
- 問題ごとに地図のズームはリセットされます
- 答え合わせ画面で正解の国は緑、誤答は赤で示されます

---

## 🗺️ 地図操作

- **ピンチ** または **マウスホイール**: 拡大/縮小（最大 100倍）
- **ドラッグ**: 地図のパン
- 右上の **+ / − / ⟲** ボタンでも操作可
- 各問題の冒頭でズームは自動リセットされます

## 💻 ローカルで動かす

ブラウザで [index.html](index.html) を直接開いても動作します
（CDN から d3 と地図データを読み込むためインターネット接続が必要）。

CORS 制約のある環境では簡易サーバ経由で:

```bash
npx serve .
# または
python -m http.server
```

## 🚀 GitHub Pages で公開する

このリポジトリをフォーク／クローンしたあと:

1. GitHub にプッシュ
2. **Settings → Pages** で
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` / **folder**: `/ (root)`
3. 数十秒で `https://<ユーザ名>.github.io/<リポジトリ名>/` で公開

## ⚠️ 既知の制限

- world-atlas 110m を使用しているため、シンガポール・リヒテンシュタインなど非常に小さい国は地図上で表示されない、もしくはタップが難しい場合があります（地図を拡大すれば対応可）
- グリーンランド・香港・プエルトリコなどの一部地域は地図には表示されますが、出題対象からは外しています（領土・特別行政区扱い）
- 一部の国境は簡略化されています（110m解像度のため）
